Fraudulent Application!

Fraudulent web applications imitate legitimate ones and pose risks like aggressive ADS, data interception, and device corruption. Users struggle to differentiate between authentic and fake apps. To address this, a platform for user queries before installation is proposed, using sentiment analysis (VADERS) to analyze ratings and comments, helping identify fraudulent apps based on user feedback.

Technologies Used
Vader's Algorithm

The VADER algorithm is employed to predict fraudulent apps by analyzing user ratings and comments. It assesses sentiment intensity, helping to identify negative feedback patterns that might indicate a fake or malicious application.


Google Play Scrapper

Google Play Scraper extracts real-time reviews and ratings from the Google Play Store. This data is analyzed using the VADER sentiment analysis algorithm to detect negative sentiment patterns, identifying potentially fraudulent apps based on user feedback. This approach ensures accurate and efficient fraud detection.

ML & Blockchain Mobility : Secure Ride-Sharing & Seamless Auto Repairs

This project integrates Machine Learning and blockchain technologies to innovate auto repairs and ride-sharing. It features a Mechanical Shop Locator using cosine similarity and vectorization for precise service recommendations based on location and reviews. Simultaneously, a decentralized Carpooling Service employs blockchain for secure, transparent transactions. Extensive testing highlights significant improvements in service accuracy and transaction security, demonstrating the project's transformative potential in transportation services.

Technologies Used
Javascript

The project uses the MERN stack, comprising MongoDB, Express.js, React, and Node.js. This technology stack enables efficient full-stack development, allowing for robust, scalable, and high-performance applications.

Blockchain

Blockchain technology ensures secure, transparent transactions in the carpooling service with its decentralized, immutable ledger. This guarantees tamper-proof data and user trust in handling credentials and payments.

Machine Learning

Vectorization techniques, cosine similarity, and NLP (Natural Language Processing) algorithms are used to train the model for the project. Vectorization converts text into numerical vectors, cosine similarity measures the similarity between these vectors, and NLP algorithms process and analyze the textual data. NLP enables understanding and interpreting human language, ensuring accurate and relevant service recommendations based on user input.
